Offline Emergency-Enabled Voice Support for Elderly Care

About the Project

The Elder Voice Command Assistant is an offline, voice-controlled system designed to assist elderly users with daily interactions and emergency situations. The system prioritizes reliability, simplicity, and safety.

Core Features

  • Offline speech recognition using Vosk
  • Emergency alert with repeated voice warnings
  • Medicine reminders and basic queries
  • Simple and elder-friendly interface
